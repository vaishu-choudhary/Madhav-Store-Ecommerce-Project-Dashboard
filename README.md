# Madhav Store Ecommerce Dashboard

## Project Overview

This project is a Power BI dashboard created to analyze Madhav Store ecommerce sales performance. The dashboard provides insights into sales, profit, quantity sold, customer trends, and product categories to help understand business performance.

## Tools Used

* Power BI
* Excel / CSV Dataset
* Data Cleaning
* Data Visualization

## Dashboard Features

* Sales Analysis
* Profit Analysis
* Category-wise Performance
* Customer Insights
* State-wise Sales Overview
* Interactive Filters and Visualizations

## Key Insights

* Identified top-performing categories and customers.
* Analyzed sales and profit trends.
* Compared performance across different states and categories.

## Files Included

* Power BI Dashboard (.pbix)
* Dataset File
* Dashboard Screenshots

## Project Objective

The main objective of this project is to transform raw ecommerce sales data into meaningful business insights using Power BI dashboards.
